# ADR-0002: Usar .NET 8 como plataforma de desarrollo

## Estado

Reemplazada por [ADR-0019](ADR-0019-nextjs-typescript.md).

## Contexto

Esta decision correspondia a la arquitectura inicial basada en API ASP.NET Core, Worker Service y cliente WPF.

## Decision anterior

La plataforma principal era .NET 8.

## Motivo del reemplazo

El proyecto se ha adaptado a una aplicacion web con Next.js y TypeScript. La decision vigente esta documentada en [ADR-0019](ADR-0019-nextjs-typescript.md).
